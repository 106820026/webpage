現貨和股票期貨的差異

▲股票族 轉 個股期貨 必知！
▲成本 操作金額 省很大！
▲輕鬆雙向操作 免借！

舉例來說鴻海股票期貨

若

買鴻海股票100張，110元 x 1000股 x 100張 = 1100萬
   (融資) 1100萬 x 0.6 = 660萬，
   (融資利率) 660萬 x 5% = 33萬，
買100張鴻海股票融資，放一年，光利息就要花33萬

買50口股票期貨等值鴻海股票100張，每個月要轉倉來回為100口，
一年共12個月轉倉，共交易1200口，成本為：
1200口 x 50手續費 = 60000元就能持有等值於100張鴻海股票
轉倉12次，成本只要60000元，遠小於股票融資利率金額。

股票期貨的優點

 股票期貨操作靈活，可做多或做空
 成本更低，所需資金少，只需要現股的13.5%資金即可操作
 低稅金，參與除權息更有利，因為股息不用列入所得課稅，也不用繳交二代健保補充費，股息股利又可以當天拿到手
 低手續費、財務槓桿高
 避險優勢，對應現股，完全避險

股票期貨與股票的差異

  股票期貨   股票
利息成本 無 無
交易稅 買賣各課契約價值的10萬分之2     賣出時的千分之3
資金成本 原始保證金比例為13.5%或16.2%(視波動度而定)     交割金額100%
放空方式 無限制     券商沒有券可以借，平盤以下不能放空的限制、強制贖回的問題
所需資金 100元 x 2000股 x 13.5% = 2萬7000     100元 x 2000股 = 20萬
功能 適合短線賺價差 適合長期投資
投資風險 1.必須每日結算，有短期資金周轉的壓力
2.槓桿倍數大，交易人有超額損失之可能
3.到期時必須強制交割，無法長期持有
4.僅需承擔整體股市之系統風險 		1.不須每日結算，較無短期資金周轉的壓力
2.槓桿倍數較小，交易人不易發生超額損失
3.無到期限制，可長期持有
4.需承擔整體股市之系統風險與個股之個別風險

股票期貨契約規格 & 交易規則

  股票期貨
契約單位 1口合約=2張股票
交割 採現金交割結算
交易時間 早上8:45~下午1:45
(到期月份的結算當日交易時間為8:45~13:30)
最後交易日 每月第三個星期三，其次一營業日為新契約的開始交易日
當月最後結算價 當日交易時間收盤前60分鐘內標的證券之算術平均價
每日結算價 當日收盤前1分鐘內所有交易之成交量加權平均價
每日漲跌幅 前一營業日結算價上下7%限制
交易月份 當月起兩個月份，另外加上3、6、9、12月中三個連續的季月
